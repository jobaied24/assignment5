Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML set or return HTML content of an element and innerText set or return only visible text content of an element and textContent return all content both visible and hidden

Question-3: What is event delegation in the DOM?

In javascript even delegation is way of attaching event listener to parent element instead of attaching event listener to individual child elements.Its bubbles up to parent element from children elements.Its allows parent element to handle event for multiple children elements dynamically

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of event propagation where an event start from target elements and going up to its ancestors untill it reach to html document
